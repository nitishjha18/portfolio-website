## Features

- **Boot sequence** — fake OS boot log with typewriter effect before the UI loads
- **Interactive terminal** — fully functional CLI interface with typed commands
- **Keyboard navigation** — arrow keys move a `>` selector through menu options
- **Typewriter output** — every command response types out line by line
- **Matrix rain** — low-opacity falling characters canvas animation in the background
- **CRT scanlines** — subtle CSS overlay for the retro monitor aesthetic
- **Social links** — github, linkedin, gmail pinned to the bottom-right of the terminal
- **Uptime counter** — live system uptime displayed in the footer
- **[clear] on every page** — one-click return to the main menu from any output
- **Fully responsive** — works on desktop and mobile
- **Zero dependencies** — no frameworks, no libraries, one optional Google Font

## Commands

| Command      | Output                          |
|--------------|---------------------------------|
| `intro`      | Name, role, location, bio       |
| `experience` | Work history at Garage Collective |
| `education`  | B.Tech @ JECRC University       |
| `projects`   | Expense Tracker, Scholar Circle, Portfolio OS |
| `skills`     | Full stack tech stack + ASCII progress bars |
| `clear`      | Clears terminal, returns to menu |
| `help`       | Shows the command menu (type manually) |
